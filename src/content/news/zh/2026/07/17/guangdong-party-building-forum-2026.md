据悉，汕头大学党委、韩山师范学院党委、嘉应学院党委拟于2026年12月在汕头大学联合举办2026年粤东三校党建研究论坛，目前正面向三所高校师生征集学术论文。

本次论坛旨在深入学习研究阐释习近平党建思想，全面落实新时代党的建设总要求和新时代党的组织路线，进一步深化粤东三校党建研究和交流，发挥各级党组织在高校高质量发展中的引领作用。

## 论坛概况

| 项目 | 详情 |
|------|------|
| 活动名称 | 2026年粤东三校党建研究论坛 |
| 主办单位 | 汕头大学党委、韩山师范学院党委、嘉应学院党委 |
| 举办时间 | 2026年12月（具体日期待定） |
| 举办地点 | 汕头大学 |
| 投稿截止 | 2026年9月28日 |
| 征集范围 | 汕头大学、韩山师范学院、嘉应学院师生 |

## 征文主题与方向

本次论坛以"深入学习贯彻习近平党建思想，以正确政绩观开创粤东高校现代化建设新局面"为总主题，设置了20个参考选题方向，涵盖以下重点领域：

| 类别 | 选题方向 |
|------|----------|
| 思想理论与实践路径 | 习近平党建思想在地方高校落地生根的理论逻辑与实践路径、高校党的创新理论深化内化转化长效机制 |
| 意识形态与从严治党 | 新时代高校意识形态阵地建设与风险防控、新时代高校全面从严治党常态化精细化建设路径 |
| 党建引领发展 | 高质量党建引领教育科技人才"三位一体"协同发展、党建引领科教产教融合及服务粤东乡村振兴 |
| 正确政绩观 | 正确政绩观融入"双带头人"培育路径、基层党建考核评价体系优化、党务干部队伍能力建设 |
| 智慧党建 | 数字化技术赋能基层党建、人工智能时代数字党建与智慧党建创新发展 |
| 学生社区与品牌 | "一站式"学生社区建设、地方高校党建品牌创建与特色党建育人实践 |
| 治理现代化 | 以党的自我革命引领高校治理现代化的实践探索与经验总结 |

参会人员可选择参考上述选题，也可结合实际自拟题目。

## 投稿要求

根据通知，参评论文需符合以下要求：

| 要求项 | 说明 |
|--------|------|
| 字数 | 一般不超过5000字 |
| 格式 | 按正式发表文章要求，需提供摘要、关键词、参考文献 |
| 文件名 | 序号＋论文＋高校名称＋姓名 |

## 评审与表彰

主办方表示，论坛将邀请相关院校党建专家和学者对提交的论文进行匿名评审，按比例遴选优秀论文在论坛上予以表彰，并邀请优秀论文作者参与分论坛主题发言。

论文以二级党组织为单位统一提交，投稿邮箱为 o_dxb@stu.edu.cn。联系人：汕头大学刘敏泉、冯迎，联系电话 0754-86502345。
