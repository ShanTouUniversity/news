据悉，为响应"七彩假期"志愿服务号召，关爱未成年群体、深化校地共建，工学院团委、敬一书院联合大华街道团工委将于7月13日至19日开展"梦起大华·星光工坊"七彩假期夏令营志愿服务活动。

活动面向大华街道青少年，每日15:00至17:30举行，地点涵盖大华街道福长社区党群服务中心、集成社区居委会和汕头市科技馆。

本次夏令营由"星华"大学生实践团和"华梦"七彩假期系列课程教学突击队承办，采用"星航任务闯关"模式，学员每日完成课程任务即可收集"星光印章"，集齐印章可兑换结业证书及专属礼包。

每日课程设有知识讲解、动手实践、互动游戏三大环节。

| 日期 | 课程名称 |
| --- | --- |
| 7月13日 | 星空起航·初遇——星际漫游与团队组建 |
| 7月14日 | 薪火相传·铸魂——红色精神与薪火故事会 |
| 7月15日 | 蓝白幻彩·润心——扎染魔法与色彩挑战 |
| 7月16日 | 光之密语·启智——电路探秘与星光贺卡 |
| 7月17日 | 花木成诗·拾趣——花草纸灯与自然手作 |
| 7月18日 | 竹贝风雅·养性——竹编造物与螺钿镶嵌 |
| 7月19日 | 星辉绽放·筑梦——科技漫游与星光展演 |

> 来源：工学院团委、敬一书院 OA 通知
