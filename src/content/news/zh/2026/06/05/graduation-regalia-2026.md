资源管理处近日发布通知，就 2026 届毕业典礼学位服的领取、使用及归还工作作出安排。

领取地点：D 座教学楼下家具仓库

| 学院 | 领取时间 |
|------|----------|
| 国际学院 | 09:00-09:30 |
| 继续教育学院 | 09:00-09:30 |
| 公共卫生学院 | 09:30-10:00 |
| 商学院（MBA） | 09:30-10:00 |
| 化学化工学院 | 10:00-11:00 |
| 长江艺术与设计学院 | 11:00-12:00 |
| 文学院 | 14:30-15:30 |
| 理学院 | 15:30-16:30 |
| 工学院 | 16:30-17:30 |

### 东海岸校区 · 6月11日

领取地点：宿舍楼 C 座一楼架空层

| 学院 | 领取时间 |
|------|----------|
| 马克思主义学院 | 10:00-11:00 |
| 长江新闻与传媒学院 | 10:00-11:00 |
| 数学与计算机学院 | 11:00-12:00 |
| 商学院 | 14:30-15:30 |
| 法学院 | 15:30-16:30 |

各学院须指派专人凭加盖院章的《各学院领取学位服清单》按时到场领取，领取后即时撤场，避免现场拥挤。学位服尺码按一定比例调配，各学院自备纸箱装取，按实需数量领取。

## 着装要求

学位服外不得加套其他服装；垂布佩戴在学位袍外，扣绊扣于第二钮扣上；内衣应着白色或浅色衬衫；男士着深色裤子，女士着深色裤子或素色裙子；男女均应着深色皮鞋；男士系领带，女士可扎领结。未授位时，流苏应垂于学位帽右前侧中部。

## 归还安排

| 归还日期 | 时间 | 地点 |
|----------|------|------|
| 6月26日 | 12:30-17:30 | 东海岸校区 · 室内体育馆羽毛球场 |
| 6月29日 | 10:00-16:30 | 桑浦山校区 · 大礼堂前厅 |

各学院须指派专人统一回收并以学院为单位归还，工作人员清点后由负责人在《各学院归还学位服清单》上签字确认。逾期未归还将视作丢失，须按原价赔偿。归还时需确保学位服干净整洁。

各学院在使用期间负责学位服统筹管理，提醒毕业生妥善保管，勿进行破坏性装饰。如有损坏或丢失，需按采购价原价赔偿。
