据悉，教务处发布2015年春季学期教学安排通知，要求各教学单位尽快制定开课计划并报送教材预订信息，以确保新学期教学工作顺利开展。

## 开课计划要点

2015年春季学期教学周数为16周。各教学单位在制定开课计划时需遵循以下原则：

| 事项 | 要求 |
|------|------|
| 实践课程 | 不宜安排须集中时间完成的实践和实习类课程 |
| 学分限制 | 每位学生选课最多不超过25学分 |
| 毕业需求 | 充分考虑2015届毕业生对课程修读需求，开设未完成必修课和专业选修课 |
| 时间分布 | 上课时间须均衡分布，避免个别时段排课过多或过少 |
| 教师资格 | 严格遵照《汕头大学本科教学兼课教师聘请与管理办法》执行 |
| 外聘教师 | 未列入学校事业编制或未纳入人事聘用制度的教学人员按外聘兼课教师对待，须填报申请表并经教务处批准后方可开课 |
| 双语/全英课程 | 须按《汕头大学本科双语/全英教学课程管理办法》进行申请 |
| 新课程 | 须按《开设新课程工作程序》进行申请 |

## 时间安排

| 截止日期 | 事项 |
|----------|------|
| 11月21日（第9周周五）前 | 通识课程、新开课、双语/全英教学及兼课教师申请报送 |
| 11月28日（第10周周五）前 | 开课计划录入学分制系统、教材信息录入 |
| 12月12日（第12周周五）前 | 2015年春季学期公益课程开课计划报送 |

## 教材选用

教务处强调，各开课课程所用教材须由学校统一订购，以杜绝违反财经纪律和商业贿赂等违纪违法问题的发生。各单位在选用外国教材时必须提高知识产权意识，避免侵犯他人著作权，不得擅自复制、使用未经授权的外文教材。

教材选用工作流程如下：

1. 任课教师根据开课计划确定所需教材
2. 经单位领导审核确认后，由教务员填写《汕头大学教材选用情况登记表》
3. 在11月28日前将教材信息录入学分制系统，核对无误后将教材与相应课程关联
4. 个别尚未安排教师的课程可由开课单位预先确定教材

讲义和誊印教材须在名称前注明"[讲义]"或"[誊印]"并注明提供资料的老师；价格待定教材须注明"[价格待定]"。教材征订相关表格已修订，各单位须使用最新格式。

> 来源：汕头大学教务处
